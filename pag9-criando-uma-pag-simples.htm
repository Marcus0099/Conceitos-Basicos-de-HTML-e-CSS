EXPLICANDO O CÓDIGO

  • HTML: A estrutura básica inclui um título <h1>, um parágrafo <p> e um botão <button>.
  • CSS: Vamos separar as partes do código para ficar mais fácil de entender:

    1. Corpo da Página (body):



      • background-color (cor de fundo): Como o nome já diz, essa linha tem a função de mudar a cor de fundo da página, e a cor escolhida foi preto (black).
      • font-family (família de fontes): Essa linha vai definir a fonte padrão da página. Existem muitas fontes que podem ser escolhidas, então é só escolher a que mais lhe agradar.
      • magin (margem): Essa linha aqui remove a margem padrão do navegador, o que permite que o conteúdo ocupe toda a área disponível.
      • padding (espaçamento interno): O padding vai adicionar um espaçamento de 20 pixels em torno do conteúdo para que não fique colado às bordas da janela.



    2. Título (h1):



      • color (cor): Aqui é bem simples, o "color" vai alterar a cor do que está dentro do <h1>, nesse caso, a cor do título será branca (white).



    3. Parágrafo (p):



      • color (cor): Assim como foi com o h1 acima, o "color" aqui vai alterar a cor do que está dentro do <p>, o que nesse caso vai ser a cor vermelha (red).
      • font-size (tamanho da fonte): O "font-size" vai alterar o tamanho da fonte para o que você escolher, nesse caso aqui, 30 pixels.



    4. Botão (.botao):



      • Seletor de Classe (.): No CSS, o ponto antes do nome indica que estamos utilizando um seletor de classe. Isso significa que estamos definindo estilos para todos os elementos que têm essa classe específica atribuída a eles. Lembrando que a classe "botao" foi o nome escolhido para agrupar os estilos aplicados no elemento do HTML. Então, quando .botao foi definido no CSS, estamos dizendo que "todos os elementos com a classe 'botao' devem ter os seguintes estilos".

      • background-color (cor de fundo): Dessa vez, a cor de fundo que será alterada é a cor do botão, com isso, ela vai mudar para cor cinza (gray).

      • color (cor): Aqui também é bem semelhante ao que foi feito com o <h1>, estamos alterando a cor da fonte para a cor branca (white).

      • padding (espaçamento interno): Com isso, o botão vai ter um espaçamento interno de 10 pixels na vertial e 15 pixels na horizontal, o que vai deixar o texto dentro do botão bem centralizado.

      • border (borda): Alguns navegadores, por padrão, aplicam bordas a botões, o que pode não combinar com o design que você planejou. Usar isso vai remover qualquer borda padrão que o botão possa ter.

      • border-radius (raio da borda ou cantos arredondados): Com a borda padrão removida, você pode escolher qual tamanho quiser. Aqui foi escolhido borda com raio de 5 pixels, o que vai deixá-la levemente arredondada.

      • cursor: O cursor é aquele ícone que aparece na tela para indicar a posição do ponteiro do mouse. Aqui, o nosso "cursor" vai alterar o formato desse ponteiro. Quando passar sobre o botão, o cursor vai mudar para uma mão, isso vai indicar que o botão é interativo.



    5. Efeito de Hover ou Passar o Mouse no Botão (.botao:hover):


      • Efeito de Hover: Quando o mouse passa sobre o botão, a sua cor de fundo muda para vermelho (red). Isso cria um feedback visual, indicando que o botão pode ser clicado:
