COMO INTEGRAR CSS E HTML

Já que o CSS ajuda a estilizar páginas na internet, podemos dizer que ele e o HTML andam juntos no desenvolvimento web. E para que um possa ajudar o outro, nesse caso, para que o CSS possa estilizar o que está no HTML, primeiro precisamos colocar ambos para trabalhar em conjunto.

Existem três maneiras principais de integrar o CSS em um documento HTML: inline, internal e external. Cada uma tem suas próprias características e usos.

  • Inline CSS: O CSS inline é aplicado diretamente em uma tag HTML usando o atributo style. Essa abordagem é útil para pequenas alterações rápidas, mas não é recomendada para estilos em larga escala, pois pode deixar o código desorganizado. Por exemplo, vamos ver um título não estilizado:

    <h1>Título Vermelho</h1>



    Agora, vamos ver esse mesmo título, porém com alguma estilização:

    <h1 style="color: red; font-size: 24px;">Título Vermelho</h1>



    Aqui, estamos estilizando o título de forma direta, o que pode ser prático, mas não muito eficiente.



  • Internal CSS:O CSS interno é definido dentro da tag <style> no cabeçalho <head> do documento HTML. Isso permite aplicar estilos a vários elementos em uma única página sem poluir o HTML. Um exemplo seria:

  • External CSS: O CSS externo é o método mais recomendado para projetos maiores. Semelhante à criação de um arquivo HTML, você vai apenas mudar o que vem depois do ponto, no lugar de .html, você vai digitar .css. Assim, um novo arquivo será criado de forma separada, tudo que você precisa fazer é o vincular ao seu HTML usando a tag <link>.
