ANTES DE COMEÇAR

Antes de mergulhar nos conceitos básicos de HTML e CSS, há algumas preparações simples que vão ajudar a tornar sua jornada mais tranquila e produtiva. Aqui estão alguns passos simples para garantir que você esteja pronto para começar:

  1. Instale um Editor de Código: O primeiro passo é ter um bom editor de código. Eu recomendo o Visual Studio Code (VS Code), que é gratuito e bastante intuitivo. Com ele, você vai conseguir escrever seu código com mais facilidade. Você pode baixar o VS Code clicando aqui. E não se preocupe se você nunca usou um editor de código antes, o VS Code é bem amigável e existem vários tutoriais rápidos e ótimos no YouTube.


  2. Configure o VS Code: Após instalar o VS Code, é uma boa ideia configurar o ambiente para ficar do seu jeito. Existem muitas extensões que podem te ajudar com isso, mas o objetivo aqui é ser prático, então eu recomendo instalar a extensão Live Server, que permite visualizar suas alterações feitas nos arquivos HTML e CSS em tempo real.


  3. Instalando o Live Server: Para te ajudar, aqui estão os passos para instalar o Live Server:
    1. No menu esquerdo, encontre a opção Extensions e clique nela.


    2. Uma nova janela vai se abrir, então, é só encontrar a barra de pesquisa.


    3. Na barra de pesquisa, digite Live Server. A extensão vai aparecer logo abaixo, é só clicar nela.


    4. Ao clicar no ícone da extensão, uma nova janela será aberta à direita. Basta encontrar a opção install. Você vai notar que nesse exemplo a opção install não existe, isso é porque eu já tenho o Live Server instalado no meu VS Code.


  4. Crie uma Pasta para seus Projetos: Organize seu trabalho criando uma pasta específica para seus projetos de HTML e CSS. Isso vai ajudar a manter tudo no lugar e facilitar o acesso aos seus arquivos.


  5. Explore Recursos Adicionais: O que será passado aqui é apenas um esboço do que o HTML e CSS oferecem, acredite, tem muito mais coisas que você poderá fazer. Então, se o que você ver aqui despertar o seu interesse, considere explorar tutoriais online, vídeos e cursos que podem complementar o que você vai aprender. A prática e a curiosidade são grandes aliadas no aprendizado de desenvolvimento web.
