TAGS E ELEMENTOS

Tags são como as placas de sinalização que indicam o que está acontecendo em uma página. Elas ficam entre os sinais < >. Por exemplo, em <p>Texto aqui</p>, temos duas placas: <p> diz "aqui começa um parágrafo" e </p> diz "aqui termina o parágrafo".

Elementos, por outro lado, são o que essas placas estão sinalizando. Eles incluem a placa de início, o que está escrito entre as placas e a placa de fim. Então, usando o mesmo exemplo, o elemento completo é <p>Texto aqui</p>. É a combinação da sinalização e do conteúdo que forma uma unidade.

Em resumo, as tags são os sinais que organizam tudo, enquanto os elementos são o que realmente vemos e interagimos na página.

ATRIBUTOS

Atributos são como pequenos rótulos que você pode adicionar às placas de sinalização (tags) para fornecer mais detalhes sobre o que está acontecendo. Eles ajudam a personalizar e definir o comportamento dos elementos na página HTML.




COMO FUNCIONAM OS ATRIBUTOS

Os atributos são colocados dentro da tag de abertura, logo após o nome da tag. Vamos usar o mesmo exemplo de antes:

A tag <p> (que é a placa de sinalização) recebe um atributo chamado class, e "editar" é o valor desse atributo. O atributo class permite que você dê a um elemento um nome que pode ser usado mais tarde para aplicar estilos e outras características. Pense nisso como um rótulo que você coloca em um grupo de elementos, facilitando a aplicação de mudanças de forma consistente. Por exemplo, se você quiser que vários parágrafos tenham a mesma cor ou fonte, basta usar o mesmo nome de classe em todos eles. Assim, fica muito mais fácil organizar e estilizar sua página.

Um atributo sempre deve ter dois componentes principais:

  1. Nome do Atributo: É a identificação que você usa para especificar qual informação ou comportamento está sendo definido. Nesse caso, o nome do nosso atributo é class.
  2. Valor: É a informação que você quer associar ao nome do atributo. O valor deve ser colocado entre aspas. Nesse exemplo é o "editar".



ESTRUTURA BÁSICA DO HTML

Um documento HTML começa com uma estrutura simples, mas fundamental para qualquer página web. Aqui está um exemplo:




EXPLICAÇÃO DA ESTRUTURA

  • <!DOCTYPE html>: Declara que este documento é HTML5 (a 5ª e mais atualizada versão do HTML).
  • <html>: Elemento raiz que contém toda a página.
  • <head>: Esta seção contém informações sobre a página que não são exibidas diretamente no conteúdo. Inclui:
    • Metatags: Como a <meta charset="UTF-8">, que define a codificação de caracteres, garantindo que os textos sejam exibidos corretamente. Outra importante é a <meta name="viewport">, que otimiza a visualização em dispositivos móveis.
    • <title>: Define o título da página, que aparece na aba do navegador e nos resultados de busca.
  • <body> É aqui que todo o conteúdo visível da página reside. Isso inclui textos, imagens, links e outros elementos interativos. Por exemplo, as tags <h1> e <p> são usadas para criar títulos e parágrafos, respectivamente.

Essas partes juntas formam a espinha dorsal de qualquer página HTML, permitindo que navegadores a interpretem e exibam corretamente.
