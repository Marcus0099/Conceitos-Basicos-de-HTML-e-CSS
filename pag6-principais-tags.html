PRINCIPAIS TAGS HTML

Você já sabe que o HTML possui tags com funções específicas, mas talvez ainda não saiba que existem centenas delas. Ler e memorizar todas pode levar muito tempo, por isso, aqui estão algumas das principais tags HTML e suas funções para facilitar sua vida:

TAGS DE TEXTO

TAGS <h1> A <h6>

  • Essas tags são usadas para títulos e subtítulos, onde <h1> representa o título mais importante e <h6> o menos importante. Por exemplo:

    <h1>Título Principal</h1>
    <h2>Subtítulo</h2>
    <h3>Subtítulo</h3>
    <h4>Subtítulo</h4>
    <h5>Subtítulo</h5>
    <h6>Subtítulo</h6>

É assim que vai ficar no seu navegador:




TAG <p>

  • Representa um parágrafo de texto. É uma das tags mais comuns e ajuda a organizar o conteúdo:

    <p>Esse é um parágrafo de exemplo.</p>

É assim que vai ficar no seu navegador:




TAG <strong>

  • É usada para destacar texto, normalmente em negrito, indicando importância:

    <p>Quero destacar um <strong>TEXTO IMPORTANTE<strong><p>

É assim que vai ficar no seu navegador:




TAG <em>

  • É usada para destacar texto, normalmente em negrito, indicando importância:

    <p>Quero deixar uma parte do <em>TEXTO ENFATIZADA<em><p>

É assim que vai ficar no seu navegador:




TAG DE INSERÇÃO DE LINK <a>

  • Esta tag cria um link para outra página ou recurso:

    <a href="https://www.exemplo.com">Clique aqui </a>

    Aqui, href é um atributo da tag <a>, que define um link. O href especifica a URL para onde o link leva.

É assim que vai ficar no seu navegador:




TAG DE INSERÇÃO DE IMAGEM <img>

  • Usada para inserir imagens:

    <a src="caminho/da/imagem.jpg" alt="Descrição da imagem">

    O atributo src define o caminho da imagem, enquanto alt fornece uma descrição alternativa, importante para acessibilidade.

TAG DE CRIAÇÃO DE LISTAS <ul> E <ol>

  • <ul> e <ol>: Representam listas não ordenadas (com marcadores) e ordenadas (numeradas), respectivamente. Dentro dessas listas, usamos <li> para definir cada item: 
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>

 
E

  <ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
  </ol>

É assim que vai ficar no seu navegador:




Essas tags são fundamentais para estruturar e formatar o conteúdo das páginas web, tornando-as mais organizadas e fáceis de ler.
