UM POUCO DA HISTÓRIA DO HMTL

O HyperText Markup Language (Linguagem de Marcação de Hipertexto), ou HTML, é uma linguagem de marcação desenvolvida em 1990 por Tim Berners-Lee, um físico do CERN, um centro de pesquisas na Suíça, que teve a ideia de criar um sistema de hipertexto na internet para trocar informações entre os pesquisadores.

O primeiro documento HTML foi criado por Berners-Lee em 1991 e continha links para outros documentos, permitindo a navegação entre eles. Essa ideia de interconectividade foi revolucionária e formou a base da web moderna.

Após a introdução do HTML, o W3C (World Wide Web Consortium) foi fundado em 1994 para desenvolver padrões para a web. O HTML evoluiu ao longo dos anos com várias versões, como HTML 2.0 (1995), HTML 3.2 (1997) e HTML 4.01 (1999). Cada versão trouxe novos elementos e funcionalidades.

Em 2008, começou o desenvolvimento do HTML5, que trouxe uma série de melhorias significativas, e a capacidade de criar aplicativos web mais complexos. HTML5 foi oficialmente recomendado em 2014.




O QUE É HIPERTEXTO?

Hipertexto é um conceito que você provavelmente já usa diariamente, mesmo que não perceba. Imagine que você está lendo um artigo na internet e encontra um link que leva a outra página com mais informações sobre um tópico que chamou sua atenção. Isso é o hipertexto.

Em vez de seguir uma sequência fixa de leitura, o hipertexto permite que você navegue entre diferentes partes do texto ou até mesmo entre diferentes documentos. É como se você tivesse um mapa interativo que te leva para onde você quer ir, sem precisar seguir um caminho linear.

Na prática, isso significa que, ao clicar em um link, você pode ser direcionado para um texto relacionado, uma imagem, um vídeo ou qualquer outro recurso que complemente ou amplie o que você está lendo. Essa forma de organizar a informação torna a navegação muito mais interessante e personalizada.

O hipertexto é a base da web que usamos hoje. Quando você está navegando na internet, está basicamente saltando de um ponto para outro em uma rede de informações conectadas, tudo graças ao hipertexto. É um pouco como ter uma conversa com um amigo que sempre te dá dicas de onde encontrar mais coisas legais baseadas no que você está interessado.

Resumindo, o hipertexto é o que torna a nossa experiência online tão dinâmica e interativa, permitindo que possamos explorar e aprender de uma forma muito mais envolvente e adaptada aos nossos interesses.




O QUE É HTML?

Voltando ao tópico principal, HTML é a linguagem básica para criar páginas na web. Imagine que você está montando um livro: o HTML seria como o esqueleto, dando estrutura às suas páginas, definindo onde as seções começam e terminam, e como o conteúdo é organizado. Ele usa "tags" para marcar diferentes partes do texto, como cabeçalhos, parágrafos e links, e também para incorporar imagens e vídeos. Sem o HTML, a web seria apenas um monte de textos sem formato, então, ele é fundamental para que os sites sejam visualmente agradáveis e funcionais.




FIQUE ATENTO!

HTML é uma linguagem de marcação, não uma linguagem de programação. Vamos voltar ao exemplo do livro: a linguagem de programação seria como um conjunto de instruções detalhadas para criar e controlar todos os aspectos do livro. Seria como se você estivesse dizendo ao computador: "Calcule este valor", "Organize essas páginas de determinada forma" ou "Desenvolva uma funcionalidade específica para o livro". Em outras palavras, você está definindo exatamente o que o computador deve fazer e como ele deve executar essas tarefas.

Já a linguagem de marcação funciona como um conjunto de etiquetas e regras que ajudam a estruturar e formatar o conteúdo do livro. É como se você estivesse dizendo: "Este texto é um título", "Este parágrafo deve estar em negrito" ou "Estas seções devem ser organizadas dessa maneira". As linguagens de marcação ajudam a organizar e exibir o conteúdo de forma clara e estruturada, sem executar cálculos ou lógicas complexas.

Então, lembre-se: a linguagem de programação cria e controla funcionalidades, e a linguagem de marcação organiza e formata o conteúdo.
