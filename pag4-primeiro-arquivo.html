Vamos aos primeiros passos para criar o seu primeiro arquivo em HTML:

ABRA O WINDOWS EXPLORER

  1. Inicie o Windows Explorer: Você pode fazer isso clicando no ícone da pasta na barra de tarefas ou pressionando a tecla Windows + E no teclado.



CRIE UMA NOVA PASTA

  1. Navegue até o local onde você quer criar a sua nova pasta. Você pode criar a pasta na área de trabalho, em "Documentos" ou em qualquer outro lugar que você preferir.
  2. Clique com o botão direito em uma área vazia da janela, e no menu que aparece, selecione Novo e depois Pasta. Você também pode fazer isso pressionando a tecla Ctrl + Shift + N no teclado.
  3. Uma nova pasta vai aparecer, aí é só dar um nome para ela. Escolha o nome que mais lhe agradar e então pressione Enter.



ABRA A PASTA NO VS CODE

  1. Abra VS Code e procure pela palavra File no canto superior esquerdo e clique nela. Ao fazer isso, um menu vai aparecer.
  2. Encontre a opção Open Folder nesse menu e clique nela.
  3. Clique na pasta que você acabou de criar.
  4. Então é só clicar em Selecionar Pasta.

Agora a pasta deve ser aberta no VS Code.




CRIE UM ARQUIVO HTML

Com a pasta aberta no VS Code, você verá um painel à esquerda com o nome da pasta. A pasta que eu criei se chama HTML & CSS, e esse painel também vai mostrar os arquivos que estão na pasta quando você criá-los.

  1. Clique com o botão direito na área vazia do painel e selecione New File.
  2. Nomeie o arquivo como index.html e pressione Enter. Isso fará com que seu arquivo HTML seja criado e aberto à direita.
  3. Na linha 1 do arquivo HTML aberto à direita, tudo que você precisa fazer é digitar ! e então pressionar Enter.

Pronto! Seu primeiro arquivo HTML acabou de ser criado. E ele será assim:




ABRA SEU ARQUIVO HTML NO NAVEGADOR

Logo, logo, você vai aprender um pouco sobre como personalizar seus arquivos HTML e, para ver o andamento dessas alterações, você deve fazer o seguinte:

  1. Com seu arquivo HTML criado, clique com o botão direito do mouse sobre ele. Um menu aparecerá ao lado assim que você fizer isso.
  2. Nesse novo meneu que acabou de aparecer, clique em Open with Live Server. E pronto, seu arquivo HTML será aberto no navegador padrão do seu computador.

    3. Com isso feito, você vai poder acompanhar as alterações no seu arquivo em tempo real.



SALVE SEU ARQUIVO HTML

Não esqueça de salvar seu trabalho! No VS Code, você pode fazer isso pressionando Ctrl + S.




AINDA TEM MAIS!
